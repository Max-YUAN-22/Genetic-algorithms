# MIA投稿检查清单

## 📋 投稿前检查清单

### ✅ 论文内容检查

#### 1. 标题和摘要
- [ ] 标题简洁明确，包含关键词
- [ ] 摘要250-300字，包含背景、方法、结果、结论
- [ ] 关键词5-8个，包含"brain tumor segmentation", "multimodal", "genetic algorithm"
- [ ] 摘要包含具体数值结果

#### 2. 引言部分
- [ ] 明确研究动机和问题陈述
- [ ] 相关工作回顾完整
- [ ] 明确贡献点（4个主要贡献）

#### 3. 方法部分

#### 4. 实验部分

#### 5. 结果部分

#### 6. 讨论和结论
- [ ] 方法优势分析
